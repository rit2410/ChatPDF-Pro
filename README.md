# ChatPDF Pro - Smart Document Assistant 🤖

A powerful AI-powered PDF analysis tool that allows users to upload PDF documents and have intelligent conversations about their content. Built with Streamlit, LangChain, and Groq LLM.

![Python](https://img.shields.io/badge/Python-3.8%2B-yellow)

## 🚀 Features

- **📄 PDF Upload & Processing**: Upload any PDF document for analysis
- **💬 Intelligent Q&A**: Ask questions about your document content
- **🧠 Context-Aware**: Maintains conversation context and memory
- **🔍 Source References**: View the exact sources from your PDF
- **⚡ Fast Responses**: Powered by Groq's high-speed inference
- **🎯 Free Embeddings**: Uses open-source sentence transformers
- **📱 Responsive UI**: Clean, modern interface built with Streamlit

## 🛠️ Tech Stack

- **Frontend**: Streamlit
- **AI Framework**: LangChain
- **LLM**: Groq (Llama 3.1 8B Instant)
- **Embeddings**: Sentence Transformers (all-mpnet-base-v2)
- **Vector Store**: FAISS
- **Memory**: Conversation Buffer Window Memory
